# Contributing to Real Page Flip Engine

First off, thank you for considering contributing to this engine! It's people like you that make it better for everyone.

## Code of Conduct
Please be respectful and professional in all interactions.

## How Can I Contribute?

### Reporting Burgs
- Use the GitHub Issue Tracker.
- Describe the bug in detail and provide steps to reproduce.
- Include your Flutter version and target platform (iOS/Android/Web).

### Suggesting Enhancements
- Open an issue with the [Feature Request] tag.
- Explain why this feature would be useful.

### Pull Requests
1. Fork the repo.
2. Create a new branch (`feature/my-feature` or `fix/my-bug`).
3. Ensure your code follows the `analysis_options.yaml` rules.
4. Run `dart format .` before committing.
5. Submit your PR with a clear description of changes.

## Development Setup
1. Clone the repo.
2. Run `flutter pub get` in the package root.
3. Use the `example/` project to test your changes.
4. Run existing tests with `flutter test`.
